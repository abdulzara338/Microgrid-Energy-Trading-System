# Microgrid Energy Trading System
A decentralized energy trading platform built on Stacks blockchain where households can trade excess solar energy for STX tokens.

## 🚀 Features

- Register as energy producer
- Create and manage energy trades
- Secure peer-to-peer energy trading
- Real-time trade status tracking
- Automated payments in STX

## 🛠️ Technical Details

- Built with Clarity smart contracts
- Comprehensive test coverage
- Secure transaction handling
- Optimized gas usage
- Input validation and access controls

## 📦 Installation

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64-glibc.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new microgrid-energy-trading && cd microgrid-energy-trading
```

3. Run tests
```bash
clarinet test
```
